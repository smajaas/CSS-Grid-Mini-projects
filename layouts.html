The CSS Grid Layout module is used to create a grid-based layout system, with the help of rows and columns it makes easier to design any webpage without using floats and positioning.A Grid container is an element on which the display: grid property is applied. It is the direct parent of all the grid items. The items of the grid container are placed inside rows and columns.The grid-area property can be used as a shorthand property for the grid-row-start, grid-column-start, grid-row-end and the grid-column-end properties.The grid-column-end property defines how many columns an item will span, or on which column-line the item will end (see example at the end of this page).
